# **App Name**: Loja Premium Drop

## Core Features:

- Dynamic Product Showcase: Showcase trending products with animated carousels and interactive cards, featuring 'Add to Cart' buttons and star ratings.
- Interactive Category Grid: Animate the display of product categories using a zoom effect and subtle lighting on hover.
- Social Proof Section: Display customer testimonials in an animated slider with real photos, names, and countries, set against a gentle gradient background.
- AI Product Recommendations: Use generative AI to show recommended products.
- Email newsletter.: Collect emails to make use of leads for future business growth.

## Style Guidelines:

- Primary color: Deep black (#1a1a1a) to create a sophisticated, modern look.
- Background color: Off-white (#f2f2f2) for a clean, contrasting backdrop.
- Accent color: Electric blue (#0099ff) to highlight key elements and calls to action.
- Headline font: 'Poppins' (sans-serif) for a clean, modern and precise look, suitable for headlines and short amounts of body text.
- Body font: 'Inter' (sans-serif) for a neutral, readable design; optimal for longer blocks of text.
- Code font: 'Source Code Pro' for any displayed code snippets.
- Clean and dynamic design inspired by modern e-commerce stores, emphasizing smooth scrolling effects and fluid spacing.
- Subtle scroll reveal animations (fade, slide, zoom) and hover effects to enhance user engagement and provide dynamic feedback.
